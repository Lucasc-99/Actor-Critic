## Getting Started



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Lucasc-99/Actor-Critic.git
   ```
2. Install Pytorch and gym with pip
```sh
   pip3 install torch
   ``
   
 ```sh
    pip3 install gym
 ``

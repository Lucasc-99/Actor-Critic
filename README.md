## Getting Started



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Lucasc-99/Actor-Critic.git
   ```
   ```sh
   cd Actor-Critic
   ```
   
2. Install Pytorch and gym with pip
```sh
   pip3 install torch
   ```
   
 ```sh
    pip3 install gym
 ```
 
3. Run the scripts
```sh
   python3 -m src.cart-pole-baseline.py
```
```sh
   python3 -m src.cart-pole-a2c.py
```

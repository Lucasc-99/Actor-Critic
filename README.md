## Installation and Running Scripts

1. Clone the repo
   ```sh
   git clone https://github.com/Lucasc-99/Actor-Critic.git
   ```
   ```sh
   cd Actor-Critic
   ```
   
2. Install Pytorch and Gym
   ```sh
   pip3 install torch
   ```
   
   ```sh
   pip3 install gym
   ```
 
3. Run scripts
   ```sh
   python3 -m src.cart-pole-baseline.py
   ```
   ```sh
   python3 -m src.cart-pole-a2c.py
   ```
